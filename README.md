# Linux VPS Swap Setup

One-click script to manage swap space on Linux VPS.

## Install

```bash
bash <(curl -Ls https://raw.githubusercontent.com/uniquMonte/swap-setup/main/install.sh)
```

## Menu Options

```
1) Add/Create Swap
2) Remove Swap
3) Install Script to System
4) Uninstall Script
5) Refresh Status
0) Exit
```
